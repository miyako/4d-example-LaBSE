## [sentence-transformers/LaBSE](https://huggingface.co/sentence-transformers/LaBSE)
**Type**: Embedding Model

LaBSE (Language-agnostic BERT Sentence Embedding) is a text embedding model developed by **Google**. The original [research paper](https://research.google/blog/language-agnostic-bert-sentence-embedding/) was published in 2020. It was trained on a vocabulary of `17` billion words in `109` languages and `6` billion pairs of English and  non-English sentences.
